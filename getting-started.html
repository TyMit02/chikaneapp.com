Getting Started with Chikane

1. Download the App
2. Create an Account
3. Set Up an Event
4. Start Timing Laps
5. Analyze Results

Step 1: Download the App

Visit the App Store to download Chikane. Launch the app and sign up to get started.

Step 2: Create an Account

Open the app and register with your email or connect through a social login option.

Step 3: Set Up an Event

Use the event management feature to create your event, add participants, and set up the event details.

Step 4: Start Timing Laps

Once your event is live, use the lap timing feature to begin tracking laps.

Step 5: Analyze Results

Access detailed lap data and analytics after the event to review performance.
